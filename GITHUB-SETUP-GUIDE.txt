═══════════════════════════════════════════════════════════════════════════
  STEP 1: CREATE THE GITHUB REPOSITORY
═══════════════════════════════════════════════════════════════════════════

1. Go to https://github.com/new

2. Fill in repository details:

   Repository name:
   ├─ streamdeck-vscode-config (recommended)
   ├─ vscode-streamdeck-ultimate
   └─ OR your own creative name

   Description:
   "Complete Stream Deck configuration for VS Code with 90 professionally 
   curated commands across 6 themed pages. Includes step-by-step setup 
   guides, printable cheat sheets, and customization tools."

   Visibility:
   ✅ Public (so others can benefit!)
   ⬜ Private (if you prefer)

   Initialize repository:
   ⬜ DON'T add README (we have one!)
   ⬜ DON'T add .gitignore (we have one!)
   ⬜ DON'T add license (we have one!)

3. Click "Create repository"


═══════════════════════════════════════════════════════════════════════════
  STEP 2: PREPARE YOUR LOCAL FILES
═══════════════════════════════════════════════════════════════════════════

1. Create a folder for your repository:
   
   mkdir streamdeck-vscode-config
   cd streamdeck-vscode-config

2. Copy all files from this package to that folder
   (Copy everything you downloaded from Claude)

3. Rename GITHUB-README.md to README.md:
   
   mv GITHUB-README.md README.md
   
   (Or on Windows: rename GITHUB-README.md README.md)

4. Edit README.md and replace:
   - "YOUR-USERNAME" with your GitHub username
   - "[Your Name]" in LICENSE with your actual name


═══════════════════════════════════════════════════════════════════════════
  STEP 3: INITIALIZE GIT AND PUSH
═══════════════════════════════════════════════════════════════════════════

In your repository folder:

1. Initialize Git:
   git init

2. Add all files:
   git add .

3. Commit:
   git commit -m "Initial commit: Complete Stream Deck VS Code configuration"

4. Link to your GitHub repository:
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/REPO-NAME.git

5. Push to GitHub:
   git push -u origin main


═══════════════════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

❌ "Permission denied (publickey)"
   → Set up SSH keys: https://docs.github.com/en/authentication
   → Or use HTTPS instead of SSH

❌ "Repository not found"
   → Check repository name spelling
   → Verify you own the repository
   → Check remote URL: git remote -v

❌ "Failed to push"
   → Pull first: git pull origin main
   → Then push: git push origin main
