
============================================================
STREAM DECK CONFIGURATION: Navigation & Files
============================================================

HOW TO USE THIS CONFIGURATION:
1. Open Stream Deck software
2. Create a new page (or use existing)
3. For each button below:
   - Drag 'Visual Studio Code: Execute Command' to button position
   - Set the title and command ID as specified
   - Optionally customize the icon
