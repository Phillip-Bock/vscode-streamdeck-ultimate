║                      FILE FORMAT SUMMARY                                   ║
╚════════════════════════════════════════════════════════════════════════════╝

  📄 .txt files  → Plain text, easy to read, copy/paste commands
  📄 .md files   → Markdown format, nice formatting, good for viewing
  🌐 .html file  → Web page, interactive, printable, beautiful
  📄 .json files → Configuration data, for modification/import
  📄 .csv file   → Spreadsheet format, for importing to other tools
  🐍 .py files   → Python scripts, for regenerating or customization

║                    QUESTIONS & TROUBLESHOOTING                             ║
╚════════════════════════════════════════════════════════════════════════════╝

  Q: Which file should I start with?
  A: README.md first, then streamdeck-setup-step-by-step.txt

  Q: Can I print something to keep on my desk?
  A: Yes! Open streamdeck-cheatsheet.html and print it

  Q: How do I find command IDs for custom buttons?
  A: VS Code → File → Preferences → Keyboard Shortcuts → Right-click command

  Q: Do I need all these files?
  A: No! Start with README.md, step-by-step.txt, and cheatsheet.html

  Q: Which file shows me ALL commands at once?
  A: streamdeck-profile-guide.txt or streamdeck-cheatsheet.html

  Q: How do I modify the configuration?
  A: Edit streamdeck-config-generator.json, then run generate_streamdeck_config.py
